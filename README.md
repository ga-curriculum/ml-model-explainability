## About
An introduction to the importance of model explainability and interpretability. 
Learners will understand why this is important, which models tend to be more explainable, and how to assess a model's explainability.

This is backed up with practical exercises and case studies.


## Learning Objectives

*You will be able to:*

- Understand what model interpretability is, and why it’s important.
- Discuss real world examples of more and less explainable models.
- Understand and implement metrics including feature importance and decision tree visualisations as examples of explainability.
